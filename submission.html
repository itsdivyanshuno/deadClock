1. Problem Statement Selected

The Last‑Minute Life Saver – Build an AI‑powered productivity companion that proactively helps users plan, prioritize, and complete tasks before deadlines.

Why this problem?

  • Directly targets the hackathon’s Agentic Depth (20%) criterion – the AI doesn’t just chat; it acts.
  • Maximizes use of Google Gemini / Google AI Studio (Gemini API + Function Calling).
  • Enables autonomous behavior that can be demonstrated live during judging.

2. Solution Overview

We built a Next.js 16 (App Router) application that couples a conversational chat interface with a powerful AI agent built on Google’s Gemini API using its Function Calling capability.

The agent maintains a persistent SQLite database of tasks and goals and can:

  • Create, prioritize, and complete tasks.
  • Set long‑term goals with milestone tracking.
  • Suggest schedules, reminders, and proactive actions based on the user’s current workload.
  • Autonomously break down high‑level goals into weekly/daily actionable tasks.
  • Reschedule tasks at risk of missing deadlines.

    • All state survives page refreshes and server restarts, giving the demo a “real‑app” feel.

    3. Key Features

    Feature Description How it shows Agentic Depth
    Chat‑Driven Task Management Users type natural‑language requests (e.g., “Add a task to finish the report by Friday”). AI parses the request and calls add_task to create the task.
    Smart Prioritization prioritize_tasks sorts pending tasks by urgency/impact. AI decides the order without user‑specified sorting.
    Proactive Suggestions suggest_proactive_actions analyses workload and returns time‑blocking tips, overdue warnings, focus‑area nudges, goal‑progress alerts, and task‑breakdown ideas. AI anticipates needs and offers unsolicited, helpful advice.
    Autonomous Goal Breakdown break_down_goal takes a goal title, deadline, and weekly availability → returns a list of concrete tasks with dates and estimates. AI plans multi‑step work on its own.
    Risk‑Based Rescheduling reschedule_at_risk_tasks flags tasks within a risk window (default 24 h) and suggests realistic new deadlines with a buffer. AI actively mitigates deadline risk.
    Goal & Milestone Tracking Users create goals; the UI shows progress % and lets them tick milestones. Provides long‑term visibility that the AI can reference in suggestions.
    Persistent Storage SQLite database (data.db) stores tasks, goals, and chat history. Enables the agent to remember past interactions and build context over time.
    Responsive UI Tailwind‑styled Next.js app with sidebar (Chat / Tasks / Goals) and celebratory feedback on completion. Delivers a polished, usable product experience.
    Deployable One‑click deploy to Vercel or Google Cloud Run. Meets the submission requirement of a public live link.

    4. Technologies Used

    Layer Choice Reason
    Frontend Next.js 16 (App Router), React 19, Tailwind CSS All‑in‑one framework (frontend + API routes), rapid UI styling, easy deployment.
    Backend / API Next.js API Routes (Node.js) No separate server; seamless integration with frontend.
    AI Engine Google Gemini API (@google/genai) + Function Calling Provides the agentic core – AI can invoke tools to act on data.
    Database SQLite (via better-sqlite3) Zero‑config persistence for a hackathon demo; faster than external DB setup.
    Dev Tools ESLint, TypeScript, PostCSS Ensures code quality and type safety.
    Deployment Vercel (or Google Cloud Run) Fast preview URLs; Vercel is fastest for Next.js, Cloud Run showcases Google tech.

    5. Google Technologies Utilized

    This section directly addresses the “Usage of Google Technologies” judging criterion – 15% of total score.

    Google Technology How We Used It
    Google AI Studio Created an account, generated the Gemini API key (stored in .env.local as GEMINI_API_KEY). This is the gateway to accessing Gemini models.
    Gemini API Integrated via the @google/genai@^2.10.0 SDK. All model calls (generateContent) route through this API, providing the conversational and reasoning backbone of the agent.
    Gemini Function Calling Implemented 9 distinct functions:
    • Original: add_task, prioritize_tasks, complete_task, add_goal, suggest_schedule, get_reminders
    • Added for agentic depth: suggest_proactive_actions, break_down_goal, reschedule_at_risk_tasks
    These functions enable the AI to act (create tasks, reschedule, break goals, suggest proactive steps) rather than merely chat.
    (Optional – not implemented) Google Authentication, Google Calendar Integration – listed as optional in the hackathon notes; omitted to keep scope focused and deliverable within the timeframe.

    By combining the Gemini API with Function Calling, the agent moves beyond a simple chatbot: it perceives user intent, selects the appropriate tool, mutates persistent state, and returns actionable results—exactly the behavior judged under “Agentic Depth.”
